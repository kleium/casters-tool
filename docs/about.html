What is Caster's Tool?

A real-time companion app for FIRST Robotics Competition broadcasters and commentators. It surfaces team stats, alliance breakdowns, playoff brackets, play-by-play data, and historical context. Everything a caster needs at a glance!

Features

Events

Season event picker with region/week filters, search, manual event code entry, and saved events for instant loading.

Rankings

Live team rankings with rank, record, OPR, DPR, CCWM. Toggle school names and select teams to compare.

Summary

Event demographics, Hall of Fame teams, Impact finalists, playoff connections graph, and top scorers.

Play by Play

Match-by-match view with per-team stats, inline comparison, and quick navigation between matches.

Breakdown

Detailed score breakdowns per match. Supports 2025 REEFSCAPE and 2026 game field layouts.

Alliances

Alliance selection cards with first-time-partner detection, partnership history, and combined stats.

Playoffs

Full double-elimination bracket visualization (2023+ format) with live score updates.

History

Event lineage — past winners, finalists, awards timeline back to 1992. Region facts & district stats.

Team Lookup

Individual team profiles with awards, blue banners, season achievements, and head-to-head playoff history.

How to Use

  • Load an event — Pick from the season list, search by name, or enter an event code manually (e.g. 2025arc).
  • Navigate tabs — Switch between Rankings, Summary, Play by Play, Breakdown, Alliances, Playoffs, and History.
  • Compare teams — Check boxes next to teams in Rankings, then click the floating compare bar (or press C).
  • Quick team lookup — Select one team and press T, or use the Team Lookup tab.
  • Save events — Click the save icon on the event bar to cache data for instant reloading.
  • Toggle settings — Use the gear icon to switch themes, highlight international teams, or show offseason events.

Data Sources

Caster's Tool pulls from two external APIs to provide comprehensive FRC data:

  • The Blue Alliance (TBA) — Event lists, team info, rankings, OPRs, match results, alliances, awards, and media.
  • FIRST FRC Events API — Score breakdowns, per-robot match performance, school names, and team avatars.

Data is cached in-memory on the server and in the browser's IndexedDB for fast reloading. Event snapshots can be saved to disk for instant access.
